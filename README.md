# EverythingCompressed

([English](#English) | [日本語](#日本語))

# 日本語

## 機械
### Everything Compressor
- 任意のアイテムを圧縮し、シンギュラリティにすることができる
- シンギュラリティ1つには元のアイテムが10,000個必要(コンフィグで変更可能)
- シンギュラリティもまた圧縮することができる
### Singularity Extractor
- シンギュラリティから元のアイテムを抽出できる
- 1つのシンギュラリティは30秒に1つのアイテムを生産する(コンフィグで変更可能)
- 抽出時にシンギュラリティは消費されない
- 複数のシンギュラリティを搬入すればその分動作速度は上がる

# English
## Machines
### Everything Compressor
- Can compress any item into a Singularity.
- Each Singularity requires 10,000 of the original item (configurable).
- Singularities can be compressed again.
### Singularity Extractor
- Can extract the original item from a Singularity.
- A single Singularity produces one item every 30 seconds (configurable).
- The Singularity is not consumed when extracting.
- Supplying multiple Singularities increases the production speed accordingly.
